Docker Desktop 설치 : https://www.docker.com/ ==> Download Docker Desktop 갖다 되면 Download for Windows - Amd64 클릭
설치 후 바탕화면 실행할때 이메일은 입력 안하고 skip해도 상관없음

wsl -l -v # wsl 목록 조회

wsl --export docker-desktop "D:\docker-desktop.tar"
wsl --unregister docker-desktop
wsl --import docker-desktop "D:\Docker" "D:\docker-desktop.tar" --version 2
